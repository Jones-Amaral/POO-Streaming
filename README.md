## Descrição

Sistema de catálogo de filmes e séries desenvolvido em C#.
A aplicação permite cadastrar, organizar e consultar conteúdos audiovisuais,
com suporte a múltiplos perfis de usuário, listas de favoritos e
organização por categorias.

Os dados são persistidos em formato TXT.

## Principais funcionalidades

- Cadastro de filmes e séries com informações detalhadas
- Suporte a temporadas e episódios para séries
- Gerenciamento de usuários com múltiplos perfis por conta
- Lista de favoritos por perfil
- Organização de conteúdo por categorias
